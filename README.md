# hotwheel-GT3

家用风火轮跑道的赛事管理网页应用(赛车/车队数据库、比赛、赛季荣誉)。

## 一键启动

- **Windows**:双击 `start.bat`
- **Mac**:双击 `start.command`

启动后会自动:关掉旧服务 → 启动服务 → 打开浏览器。网页**底部有二维码**,手机连**同一 WiFi**扫码即可访问录入。

### 注意
- **首次启动需联网**(自动安装依赖);之后可离线使用。
- Windows 没装 Python 时,脚本会提示并打开下载页 —— 安装时务必勾选 **Add Python to PATH**。
- Windows 首次启动会弹防火墙提示,要点**允许访问**,手机才能连上。
- **关闭命令行窗口 = 停止服务**。
- 数据都在 `data/` 文件夹(`hotwheel.db` + 图片);换电脑时复制这个文件夹即可。`.venv` 不要跨系统复制(从 Mac 拷到 Windows 时先删掉它)。

## 手动启动(开发)
```
uvicorn app.main:app --host 0.0.0.0 --port 8000
```
测试:`pytest`
